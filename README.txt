		Установка
		Зайти в корневой каталог с проектом
		 1.pip install -r requirements.txt
		 2.python manage.py migrate (Если хотите создать пустую базу)
		 3.python manage.py runserver
		 4.Сервер должен запускаться localhost:8000 порт, так как в социальных системах именно такой прописан домен для редиректа и приема аутентификации.
